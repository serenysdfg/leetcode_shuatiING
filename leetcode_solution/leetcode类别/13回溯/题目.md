22. 括号生成
```python
        ans = []
        def dp(s ,l,r):
            if len(s) == n*2: #结束
                ans.append(s)
                return
            else:
                if l < n:
                    dp(s+"(",l+1,r)
                if r < l:
                    dp(s+")",l,r+1)
        
        
        dp("",0,0)
        return ans
```
